Privacy Policy for ChatApp

Last updated: 23 June 2025

1. Data Collection

We collect and store the following data:

  • Email address used for login (via Firebase Authentication)
  • Firebase Cloud Messaging (FCM) token to deliver notifications

2. Data Usage

The data is used only for authentication, identifying chat users, and sending real-time push notifications through Firebase and a secure external server (Railway).

3. Notifications

We collect FCM tokens to send push notifications. Tokens are stored securely in Firebase Firestore and used via Railway backend server to trigger real-time messages.

4. Data Deletion

You can request account deletion at any time by emailing us at o.haroon111@gmail.com. Your Firebase account and related data will be deleted within 7 business days.

5. Logout

Logging out ends your session locally. No data is deleted until requested.

6. Third-party Services

We use the following services:

  • Firebase Authentication
  • Firebase Firestore
  • Firebase Cloud Messaging
  • Railway (for push notification backend)

See their privacy policies:

  • Firebase Privacy Policy
  • Railway Privacy Policy
